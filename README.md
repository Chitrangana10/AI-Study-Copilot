# 🎓 AI Study Copilot

> Learn smarter, revise faster, and never study alone.

AI Study Copilot is an intelligent learning platform designed to transform the way students study. It leverages AI to generate notes, summaries, quizzes, flashcards, and personalized explanations, helping learners understand concepts more effectively and prepare for exams with confidence.

---

## ✨ Features

### 🤖 AI-Powered Study Assistant
Ask questions in natural language and receive instant explanations tailored to your learning level.

### 📝 Smart Note Generation
Convert lengthy study materials into concise, structured notes.

### 📚 Intelligent Summaries
Generate chapter-wise and topic-wise summaries for quick revision.

### 🎯 Quiz & MCQ Generation
Create practice tests automatically from your study content.

### 🗂️ Flashcard Creation
Generate flashcards for active recall and long-term retention.

### 💬 Interactive Learning Chat
Get personalized answers, explanations, and guidance while studying.

### 📈 Personalized Learning Experience
Receive AI-generated content adapted to your learning needs.

---

## 🚀 Why AI Study Copilot?

Traditional studying often involves:
- Reading lengthy notes repeatedly
- Creating revision material manually
- Searching multiple sources for explanations
- Spending hours preparing quizzes and flashcards

AI Study Copilot automates these tasks, allowing students to focus on understanding concepts instead of creating study material.

## ⚡ Getting Started

### Clone the Repository

```bash
git clone https://github.com/your-username/AI-Study-Copilot.git
cd AI-Study-Copilot
```

### Install Dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env` file:

```env
DATABASE_URL=your_database_url
DIRECT_URL=your_direct_database_url
GEMINI_API_KEY=your_gemini_api_key
JWT_SECRET=your_jwt_secret
```

### Run Development Server

```bash
npm run dev
```

### Deploy

```bash
npm run deploy
```
